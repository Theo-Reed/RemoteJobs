# Remote Jobs - 微信小程序
连接远程机会，AI 助力简历精准匹配。
[English Version](./README.md)

## 核心亮点：AI 简历自动裁剪

本项目最大的特色是集成了强大的 **AI 自动裁剪** 技术。不再是千篇一律的投递，而是根据您看中的每一个岗位，由 AI 自动调整简历侧重点，确保您的经历与职位描述（JD）完美契合。

## 功能特性

- ✂️ **一键 AI 裁剪**: 智能分析职位需求，自动改写简历内容，提升面试邀约率。
- 🔍 **精准职位发现**: 汇聚全球优质远程开发、设计、运营等职位。
- 📄 **多语言简历画像**: 构建专业的中英文简历，支持实时预览与编辑。
- 🖨️ **专业 PDF 导出**: 经过 AI 优化后的简历可直接导出为排版精美的 PDF。
- 🌟 **个性化职位收藏**: 快速保存心仪职位，跟踪您的求职进度。
- 📱 **原生流畅体验**: 采用微信小程序原生开发，响应速度快，交互体验佳。

## 技术栈

- **框架**: 微信小程序原生框架
- **开发语言**: TypeScript
- **样式**: WXSS (响应式布局)
- **AI 交互**: 后端集成 Google Gemini 引擎

## 目录结构

- `miniprogram/pages/`: 包含首页、简历编辑、职位详情等核心页面。
- `miniprogram/components/`: 自定义 UI 组件库。
- `miniprogram/utils/`: 封装了请求、国际化、格式化等工具函数。
